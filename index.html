Cloud Computing


Overview

Cloud Computing is one of the fastest growing areas of Information Technology. It is defined as a means of storing and accessing data over the internet instead of on your computer's hard disk.
There are three different service modules of Cloud computing: Software as a Service, Platform as a Service, and Infrastructure as a Service.

Software as a Service

As described in the CompTIA Network Security textbook, Software as a Service is where the application vendor provides access to their software running in the cloud, which is then accessed via a web browser. A well-known example would be Google Apps.

Platform as a Service

This service model gives the user control to what applications are being ran on their systems. While the cloud provider maintains control over the storage and the underlining software being ran.

Infrastructure as a Service

In this model the consumer has control over everything except the underlining infrastructure that allows the cloud to be ran.
